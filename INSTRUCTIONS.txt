# FANTASY GOLF APP - SETUP & USAGE INSTRUCTIONS

## WHAT YOU HAVE

You now have a complete Fantasy Golf app system that reads data from your Excel file.

**Files you received:**
1. `fantasy-golf-app-updated.html` - Your web app
2. `FG_Logo_Updated.png` - Your league logo
3. `league-data.json` - Current league data extracted from Excel
4. `update_league_data.py` - Python script to update the data
5. `INSTRUCTIONS.txt` - This file

---

## HOW IT WORKS

```
Your Excel File (FG_2026_AI_Scores.xlsx)
           ↓
Run Python Script (update_league_data.py)
           ↓
Generates league-data.json
           ↓
Upload to web hosting
           ↓
App automatically displays current data
```

## INITIAL SETUP (One-time)

### Step 1: Get Python installed
- **Windows:** Download from python.org and install
- **Mac:** Python is already installed (open Terminal and type `python3 --version` to confirm)

### Step 2: Install required Python library
Open Terminal (Mac) or Command Prompt (Windows) and run:
```
pip install openpyxl
```

### Step 3: Organize your files
Put these files in the SAME FOLDER:
- FG_2026_AI_Scores.xlsx (your Excel file)
- update_league_data.py (the Python script)

### Step 4: Upload to web hosting
You need a place to host your app. Options:
- **Easiest:** Netlify Drop (netlify.com/drop) - drag and drop files
- **Also easy:** GitHub Pages (free)
- **Traditional:** Your own web hosting

Upload these 3 files:
- fantasy-golf-app-updated.html (rename to index.html)
- FG_Logo_Updated.png
- league-data.json

## WEEKLY WORKFLOW (30 seconds each week)

### Every time you update scores in Excel:

1. **Update your Excel file** (FG_2026_AI_Scores.xlsx)
   - Add new tournament scores
   - Save the file

2. **Run the Python script**
   - **Windows:** Double-click `update_league_data.py`
   - **Mac:** Open Terminal, navigate to the folder, run `python3 update_league_data.py`
   
   You'll see:
   ```
   ============================================================
   FANTASY GOLF - League Data Extractor
   ============================================================
   
   Reading FG_2026_AI_Scores.xlsx...
   ✓ Extracted 10 teams
   ✓ Extracted 8 transactions
   ✓ Found 16 available players
   ✓ Saved to league-data.json
   
   SUCCESS! Your league-data.json file is ready.
   ```

3. **Upload the new league-data.json**
   - Go to your web hosting
   - Replace the old league-data.json with the new one
   - That's it! App is updated.

---

## MANAGING DROP/ADDS

When owners want to drop/add players:

1. They email or text you with their request
2. You update the "Drop-Add Log" sheet in Excel:
   - Add the date
   - Add the team name
   - Add dropped player
   - Add added player
   - Transaction # auto-calculates
3. You also update the team's roster in the "1st Half" sheet
4. Run the Python script
5. Upload the new league-data.json

The app will show:
- Updated rosters
- Updated moves used count
- Transaction history

## TROUBLESHOOTING

**"Could not find FG_2026_AI_Scores.xlsx"**
→ Make sure the Python script and Excel file are in the same folder

**"Module openpyxl not found"**
→ Run: `pip install openpyxl` (or `pip3 install openpyxl` on Mac)

**App shows old data**
→ Make sure you uploaded the NEW league-data.json file

**App says "Unable to Load League Data"**
→ Check that league-data.json is uploaded to the same location as your HTML file

## IMPORTANT NOTES

### What the app DOES:
- ✓ Displays current standings
- ✓ Shows team rosters
- ✓ Lists available players
- ✓ Shows transaction history
- ✓ Updates automatically when you upload new league-data.json

### What the app DOES NOT DO:
- ✗ Does not allow owners to make roster changes directly (READ-ONLY)
- ✗ Does not automatically pull scores from PGA Tour
- ✗ Does not update itself - you must run the script

This is BY DESIGN to keep you in control of the league.
